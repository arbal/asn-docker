# asn-docker
Dockerized ASN Lookup Tool and Traceroute Server ([nitefood/asn](https://github.com/nitefood/asn))

**Usage:**

`docker run -it arbal/asn [-d] HOST`

**Example:**

`docker run -it arbal/asn 8.8.8.8`
